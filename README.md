# Initiative Tracker

A simple D&D initiative tracker built with Next.js, Tailwind CSS, and Hugeicons.

## Setup

Requires **Node.js 18.17+** (Next.js 14). Your current Node 18.18.2 is supported.

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` — Start dev server
- `npm run build` — Production build
- `npm run start` — Start production server
- `npm run lint` — Run ESLint
- `npm run deploy` — Deploy to Vercel (requires Vercel CLI)

## Deploy

Connect the repo to [Vercel](https://vercel.com) or run `npm run deploy`.
